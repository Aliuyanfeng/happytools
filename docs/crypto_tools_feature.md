# 加密工具功能说明

## 功能概述

本功能在 HappyTools 的工具盒子模块中新增了加密工具，包含四个主要功能：
1. **Base64 编解码** - 支持文本的 Base64 编码和解码
2. **哈希计算** - 支持 MD5、SHA-1、SHA-256、SHA-512 哈希值计算
3. **UTF-8 编解码** - 支持文本与十六进制 UTF-8 编码之间的转换
4. **批量处理** - 一次性计算文本的所有编码和哈希值

## 实现细节

### 1. 工具函数库 (frontend/src/utils/cryptoUtils.ts)

#### Base64 编解码功能
- `base64Encode()`: Base64 编码函数，支持 Unicode 字符串
- `base64Decode()`: Base64 解码函数，支持 Unicode 字符串
- `isValidBase64()`: 验证字符串是否为有效的 Base64 编码

#### 哈希计算功能
- `md5Hash()`: 计算 MD5 哈希值（十六进制）
- `sha1Hash()`: 计算 SHA-1 哈希值（十六进制）
- `sha256Hash()`: 计算 SHA-256 哈希值（十六进制）
- `sha512Hash()`: 计算 SHA-512 哈希值（十六进制）
- `computeAllHashes()`: 批量计算所有哈希值

#### UTF-8 编解码功能
- `utf8Encode()`: 将字符串编码为 UTF-8 字节数组（十六进制表示）
- `utf8Decode()`: 将十六进制 UTF-8 字节数组解码为字符串
- `isValidHex()`: 验证字符串是否为有效的十六进制

#### 辅助功能
- `copyToClipboard()`: 复制文本到剪贴板
- `formatHex()`: 格式化十六进制字符串（每2个字符加一个空格）
- `computeAllEncodings()`: 批量计算所有编码和哈希值

### 2. 用户界面组件 (frontend/src/components/CryptoTools.vue)

#### 组件结构
- **Base64 编解码标签页**: 提供文本的 Base64 编码和解码功能
- **哈希计算标签页**: 提供多种哈希算法的计算功能
- **UTF-8 编解码标签页**: 提供文本与 UTF-8 十六进制编码的转换功能
- **批量处理标签页**: 提供一次性计算所有编码和哈希值的功能

#### 主要功能
1. **实时计算**: 输入文本后点击相应按钮进行计算
2. **结果展示**: 以友好的方式展示计算结果
3. **复制功能**: 一键复制计算结果到剪贴板
4. **格式化功能**: 格式化十六进制字符串以便阅读
5. **批量处理**: 一次性计算所有编码和哈希值
6. **错误处理**: 提供友好的错误提示信息

### 3. 集成到工具盒子 (frontend/src/views/Toolbox/Toolbox.vue)

- 将 CryptoTools 组件集成到"加密工具"标签页（第二个标签页）
- 更新了工具箱介绍，添加了加密工具说明
- 保持了原有的导航结构

## 使用说明

### Base64 编解码
1. 在"加密工具"标签页选择"Base64 编解码"子标签
2. 在文本框中输入要编码或解码的文本
3. 点击"Base64 编码"按钮进行编码
4. 点击"Base64 解码"按钮进行解码（仅对有效的 Base64 编码有效）
5. 结果将显示在下方，可以点击"复制"按钮复制结果

### 哈希计算
1. 在"加密工具"标签页选择"哈希计算"子标签
2. 在文本框中输入要计算哈希值的文本
3. 可以选择以下操作：
   - 点击"MD5"按钮计算 MD5 哈希值
   - 点击"SHA-1"按钮计算 SHA-1 哈希值
   - 点击"SHA-256"按钮计算 SHA-256 哈希值
   - 点击"SHA-512"按钮计算 SHA-512 哈希值
   - 点击"计算所有哈希值"按钮一次性计算所有哈希值
4. 计算结果将以列表形式显示，每个结果包含：
   - 哈希类型标签
   - 哈希值（十六进制）
   - 操作按钮（复制、格式化、删除）
5. 可以点击"清除所有结果"按钮清除所有计算结果
6. 可以点击"复制所有结果"按钮复制所有计算结果

### UTF-8 编解码
1. 在"加密工具"标签页选择"UTF-8 编解码"子标签
2. 在文本框中输入要编码或解码的文本
3. 点击"UTF-8 编码（转十六进制）"按钮进行编码
4. 点击"UTF-8 解码（十六进制转文本）"按钮进行解码（仅对有效的十六进制字符串有效）
5. 结果将显示在下方，可以点击"复制"按钮复制结果
6. 可以点击"格式化"按钮格式化十六进制字符串
7. 查看"UTF-8 示例"了解常见文本的 UTF-8 编码

### 批量处理
1. 在"加密工具"标签页选择"批量处理"子标签
2. 在文本框中输入要处理的文本
3. 点击"批量处理所有编码和哈希"按钮
4. 结果将以折叠面板形式显示，包含：
   - Base64 编码
   - UTF-8 编码（十六进制）
   - MD5 哈希值
   - SHA-1 哈希值
   - SHA-256 哈希值
   - SHA-512 哈希值
5. 可以展开/折叠各个结果面板
6. 可以点击每个结果旁边的复制按钮复制单个结果
7. 可以点击"复制所有结果"按钮复制所有结果
8. 可以点击"清除结果"按钮清除所有结果

## 技术实现

### Base64 编解码
使用浏览器的 `TextEncoder` 和 `TextDecoder` API 处理 Unicode 字符串，确保正确处理中文等非 ASCII 字符。

### 哈希计算
使用浏览器的 `crypto.subtle.digest()` API 计算哈希值，支持以下算法：
- MD5
- SHA-1
- SHA-256
- SHA-512

### UTF-8 编解码
使用 `TextEncoder` 和 `TextDecoder` API 进行 UTF-8 编码和解码，支持完整的 Unicode 字符集。

### 剪贴板操作
使用现代的 `navigator.clipboard.writeText()` API，并提供备用方案使用 `document.execCommand('copy')` 确保兼容性。

## 安全注意事项

1. **哈希函数安全性**:
   - MD5 和 SHA-1 已不再安全，不应用于密码存储等安全场景
   - SHA-256 和 SHA-512 目前是安全的哈希算法
   - 建议使用加盐的密码哈希函数（如 bcrypt、scrypt、Argon2）进行密码存储

2. **Base64 安全性**:
   - Base64 不是加密算法，只是编码方式
   - 不要使用 Base64 存储敏感信息

3. **UTF-8 安全性**:
   - 确保输入验证，防止注入攻击
   - 处理用户输入时注意字符编码问题
