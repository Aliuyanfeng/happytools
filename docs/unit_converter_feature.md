# 单位转换功能说明

## 功能概述

本功能在 HappyTools 的工具盒子模块中新增了单位转换工具，包含三个主要功能：
1. **字节转换** - 支持 B、KB、MB、GB、TB、PB、EB 之间的转换
2. **时间转换** - 支持毫秒、秒、分钟、小时、天、周、月、年之间的转换
3. **时间差值计算** - 计算两个日期时间之间的差值，支持多种格式输入

## 实现细节

### 1. 工具函数库 (frontend/src/utils/unitConverter.ts)

#### 字节转换功能
- `BYTE_UNITS`: 字节单位定义数组
- `convertBytes()`: 字节单位转换函数
- `formatBytes()`: 格式化字节数为人类可读字符串

#### 时间转换功能
- `TIME_UNITS`: 时间单位定义数组
- `convertTime()`: 时间单位转换函数
- `formatTime()`: 格式化时间为人类可读字符串

#### 时间差值计算功能
- `calculateTimeDifference()`: 计算两个日期之间的时间差
- `parseTimeString()`: 解析时间字符串为 Date 对象
- `formatDateTime()`: 格式化日期时间为字符串

### 2. 用户界面组件 (frontend/src/components/UnitConverter.vue)

#### 组件结构
- **字节转换标签页**: 提供字节单位之间的实时转换
- **时间转换标签页**: 提供时间单位之间的实时转换
- **时间差值计算标签页**: 提供日期时间差值计算

#### 主要功能
1. **实时转换**: 输入数值或选择单位时自动计算转换结果
2. **单位交换**: 一键交换源单位和目标单位
3. **快速选择**: 提供常用单位的快速选择按钮
4. **多种输入方式**: 支持日期选择器、时间选择器和文本输入
5. **详细结果展示**: 显示完整的差值分解结果

### 3. 集成到工具盒子 (frontend/src/views/Toolbox/Toolbox.vue)

- 将 UnitConverter 组件集成到"数据转换"标签页
- 更新了其他标签页的占位内容
- 添加了工具箱功能介绍

## 使用说明

### 字节转换
1. 在"数据转换"标签页选择"字节转换"子标签
2. 输入要转换的数值
3. 选择源单位（如 KB）
4. 选择目标单位（如 MB）
5. 转换结果将实时显示
6. 可以使用"交换"按钮快速交换源单位和目标单位
7. 可以使用底部的快速选择按钮直接选择常用单位

### 时间转换
1. 在"数据转换"标签页选择"时间转换"子标签
2. 输入要转换的数值
3. 选择源单位（如 秒）
4. 选择目标单位（如 小时）
5. 转换结果将实时显示
6. 可以使用"交换"按钮快速交换源单位和目标单位
7. 可以使用底部的快速选择按钮直接选择常用单位

### 时间差值计算
1. 在"数据转换"标签页选择"时间差值计算"子标签
2. 设置开始时间：
   - 使用日期选择器选择开始日期
   - 使用时间选择器选择开始时间
   - 或直接输入日期时间字符串（格式：YYYY-MM-DD HH:mm:ss）
3. 设置结束时间：
   - 使用日期选择器选择结束日期
   - 使用时间选择器选择结束时间
   - 或直接输入日期时间字符串（格式：YYYY-MM-DD HH:mm:ss）
4. 点击"计算差值"按钮或自动计算
5. 查看详细结果：
   - 总毫秒数
   - 格式化结果（如：1天2小时30分钟）
   - 详细的差值分解（年、月、天、小时、分钟、秒、毫秒）
   - 总天数、总小时数、总分钟数统计

### 快捷操作
- **开始时间设为现在**: 将开始时间设置为当前时间
- **结束时间设为现在**: 将结束时间设置为当前时间
- **交换时间**: 交换开始时间和结束时间

## 技术实现

### 单位转换算法
所有单位转换都基于基准单位：
- **字节转换**: 以字节(B)为基准，其他单位通过乘以相应的因子进行转换
- **时间转换**: 以毫秒(ms)为基准，其他单位通过乘以相应的因子进行转换

### 时间差值计算
1. 将两个日期时间转换为毫秒时间戳
2. 计算时间戳差值得到总毫秒数
3. 将总毫秒数分解为各个时间单位
4. 构建人类可读的格式化字符串

### 输入解析
支持多种时间格式：
- `YYYY-MM-DD HH:mm:ss` - 完整日期时间
- `YYYY-MM-DD` - 仅日期
- `HH:mm:ss` - 仅时间（使用当前日期）
- `HH:mm` - 简单时间格式（使用当前日期）

## 示例

### 字节转换示例
- `1024 KB = 1 MB`
- `1 GB = 1024 MB = 1048576 KB = 1073741824 B`

### 时间转换示例
- `3600 秒 = 60 分钟 = 1 小时`
- `1 天 = 24 小时 = 1440 分钟 = 86400 秒`

### 时间差值计算示例
开始时间: `2026-02-10 09:00:00`
结束时间: `2026-02-12 15:30:45`
结果: `2天6小时30分钟45秒`

## 注意事项

1. **精度处理**: 转换结果保留6位小数，避免精度丢失
2. **输入验证**: 对用户输入进行验证，防止无效数据
3. **错误处理**: 提供友好的错误提示信息
4. **性能优化**: 使用防抖技术避免频繁计算
5. **用户体验**: 提供实时反馈和快捷操作
